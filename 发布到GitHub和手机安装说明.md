# 发布到 GitHub 和手机安装说明

## 1. GitHub 上传

解压后进入 `knowledge-pwa-github-upload` 文件夹，全选里面所有文件上传到仓库根目录。

这个版本没有复杂文件夹，所有 `mindmap-xxx.png`、`mindmap-xxx.pdf`、`app.js`、`index.html` 都在同一层。

## 2. 开启 GitHub Pages

进入仓库：

Settings → Pages → Branch 选择 `main`，目录选择 `/root`，保存。

## 3. 手机安装

用手机浏览器打开 GitHub Pages 地址，选择“添加到主屏幕”。

## 4. 数据保护

你的学习记录保存在手机浏览器本地。换手机或清缓存前，请先点“导出学习记录”。
