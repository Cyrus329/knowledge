# 专升本知识点速记本 v18 手机修复版

这版针对你截图里的问题修复：

- 修复考点必背原图在手机/GitHub Pages 中显示不出来的问题：把录屏资源目录改成英文路径。
- 填空背诵改成更紧凑排版：字号缩小、行距缩小、卡片不再一块一块占满屏。
- 填空输入框加宽：长答案输入后不容易显示不全。
- 明显乱码会被过滤；识别不稳定的页面不强行显示乱字，以原图答案核对。
- 考点必背仍然来自录屏黄色重点，不是 PDF 原文。
- 旧版进度键不变，可继续导入原来的学习记录。

## 上传 GitHub 注意

必须上传整个文件夹，尤其是：

- `exam_video_v15/`
- `images/`
- `pdfs/`
- `app.js`
- `knowledge-data.js`
- `styles.css`
- `index.html`

不能只上传 index.html，否则图片一定会坏。
