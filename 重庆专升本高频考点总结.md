# 高频必考01：图灵、冯·诺依曼、ENIAC
- 图灵机：抽象数学模型，不是真实机器，用来研究什么问题可以被计算。
- 通用图灵机：一台机器通过更换程序可以模拟其他图灵机，是现代通用计算机思想基础。
- 图灵测试：通过人机对话判断机器是否表现出类似人的智能。
- 图灵奖：ACM 设立，常被称为计算机领域最高奖。
- 冯·诺依曼思想：存储程序、程序控制、二进制表示、五大组成部分。
- ENIAC：通常作为世界上第一台通用电子数字计算机考。

# 高频必考02：计算机发展阶段
- 第一代：电子管计算机。
- 第二代：晶体管计算机。
- 第三代：中小规模集成电路计算机。
- 第四代：大规模和超大规模集成电路计算机。
- 发展趋势常背：巨型化、微型化、网络化、智能化、多媒体化。

# 高频必考03：计算机特点、分类、应用
- 计算机特点：运算速度快、计算精度高、存储容量大、逻辑判断能力强、自动化程度高、通用性强。
- 按处理对象/工作原理：数字计算机、模拟计算机、数模混合计算机。
- 按用途：通用计算机、专用计算机。
- 按规模/性能：巨型机、大型机、小型机、微型机等。
- 常见应用：科学计算、数据处理、过程控制、计算机辅助系统、人工智能、网络通信、多媒体应用。

# 高频必考04：系统软件与应用软件

# 高频必考05：操作系统、程序设计语言、数据库
- 操作系统功能：处理器管理、存储管理、设备管理、文件管理、作业/用户接口管理。
- 常见操作系统：Windows、Linux、Unix、macOS、Android、iOS。
- 程序设计语言：机器语言、汇编语言、高级语言。

- 数据库管理系统 DBMS 用于组织、存储、管理和检索数据。

# 高频必考06：主板、接口、总线与 I/O
- I/O 接口：连接主机与外部设备，实现数据传送和控制。
- 常见接口：USB、HDMI、VGA、网卡接口、音频接口、SATA、PCI/PCIe 等。
- 总线可理解为传送信息的公共通道，常见有数据总线、地址总线、控制总线。
- 主板常见组成：CPU 插槽、内存插槽、芯片组、BIOS/UEFI、扩展槽、外设接口。

# 高频必考07：CPU、存储器、输入输出设备
- CPU 主要由运算器和控制器组成，内部还有寄存器。
- 运算器负责算术运算和逻辑运算；控制器负责取指令、分析指令、发控制信号。
- RAM：随机存取存储器，可读可写，断电后数据通常丢失。
- ROM：只读存储器，断电后内容不易丢失。
- Cache：高速缓冲存储器，用于缓解 CPU 与主存速度差。
- 输入设备：键盘、鼠标、扫描仪、摄像头、麦克风等。
- 输出设备：显示器、打印机、绘图仪、音箱、投影仪等。

# 高频必考08：数制、编码、单位换算
- 位 bit 是最小信息单位；字节 Byte 通常 1B=8bit。
- 1KB=1024B，1MB=1024KB，1GB=1024MB，1TB=1024GB。
- 十进制转二进制：整数除 2 取余倒排，小数乘 2 取整顺排。
- 二进制、八进制、十六进制可按 3 位/4 位二进制分组转换。
- ASCII 基本码通常用 7 位二进制表示，可表示 128 个字符。
- 汉字编码常考输入码、机内码、字形码等概念。

# 高频必考10：最容易出错的判断句
- 图灵机是抽象模型，不是真实机器。
- 通用图灵机思想对应现代通用计算机，不是只能解决一个问题。
- 操作系统属于系统软件，不属于普通应用软件。
- Photoshop 主要是图像处理软件，不是文字处理软件。
- AutoCAD 常用于绘图/工程制图，不是办公文字处理软件。
- RAM 断电一般丢失，ROM 断电内容不易丢失。
- 1B=8bit，1KB 通常等于 1024B。
- 输入输出设备按信息方向判断，不按设备外观判断。

# 高频必考11：总线、主板、I/O接口
- 总线是计算机各部件之间传送信息的公共通道。
- 总线按传输信息可分为数据总线、地址总线、控制总线。
- 数据总线传数据，地址总线传地址，控制总线传控制信号。
- I/O接口连接主机和外设，负责数据交换、状态控制、格式转换等。
- 主板是计算机各部件连接和协同工作的基础平台。

# 高频必考12：CPU组成与性能指标
- CPU 主要由运算器和控制器组成，内部还有寄存器、高速缓存等。
- 运算器进行算术运算和逻辑运算。
- 控制器负责取指令、分析指令、发出控制信号。
- CPU 性能常看主频、字长、核心数、缓存容量、指令系统等。
- 字长通常指 CPU 一次能处理的二进制位数，字长越长，处理能力和精度通常越高。

# 高频必考13：存储器层次、RAM、ROM、Cache
- 存储体系常见层次：寄存器、Cache、内存、外存。
- RAM 可读可写，断电后数据通常丢失。
- ROM 只读或主要用于固化程序，断电后内容不易丢失。
- Cache 是高速缓冲存储器，用于缓解 CPU 与内存速度差。
- 内存包括 RAM、ROM 等；外存包括硬盘、U盘、光盘等。

# 高频必考14：外存储器与常见存储介质
- 硬盘、固态硬盘、U盘、光盘、移动硬盘等属于外存储器。
- 外存断电后数据通常仍可保存。
- 机械硬盘依靠磁介质存储，固态硬盘多基于闪存。
- 光盘常见类型有 CD、DVD、BD 等。

# 高频必考15：输入设备、输出设备、输入输出设备
- 输入设备：键盘、鼠标、扫描仪、摄像头、麦克风、条形码阅读器等。
- 输出设备：显示器、打印机、绘图仪、音箱、投影仪等。
- 触摸屏通常既可作为输入设备，也可作为输出设备。
- I/O 设备泛指输入输出设备。

# 高频必考16：ASCII码、字符编码、汉字编码
- ASCII 基本码通常用 7 位二进制表示，可表示 128 个字符。
- 一个字节是 8 位，ASCII 存储时常占 1 个字节。
- 汉字编码常见：输入码、国标码、机内码、字形码。
- Unicode 是统一字符编码标准，能表示多种语言字符。

# 高频必考17：进制转换与存储单位计算
- 1B=8bit。
- 1KB=1024B，1MB=1024KB，1GB=1024MB，1TB=1024GB。
- 二进制转八进制：从小数点向两边每 3 位一组。
- 二进制转十六进制：从小数点向两边每 4 位一组。
- 十进制整数转二进制：除 2 取余倒排。
